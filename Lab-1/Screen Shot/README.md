## Overview
This lab demonstrates how to create and configure a Linux Virtual Machine using KVM/QEMU on a local system.

## Steps Performed

1. Installed and opened Virtual Machine Manager.
2. Created a new virtual machine using an Ubuntu 24.04 ISO.
3. Allocated CPU, RAM, and storage.
4. Completed the Ubuntu installation:
   - Set hostname
   - Created user account
   - Configured storage (LVM)
   - Enabled network
5. Removed installation ISO from the VM.
6. Booted into the installed Ubuntu system.
7. Logged in via console successfully.
8. Enabled and started SSH service inside the VM.
9. Connected to the VM using SSH from the host system.
10. Verified:
    - IP address
    - OS version
    - Disk usage
    - Memory usage
